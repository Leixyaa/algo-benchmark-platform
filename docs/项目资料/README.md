# 项目资料索引

本目录用于沉淀工程可复用资料，保留与运行、接口、数据存储和算法接入直接相关的轻量文档。
论文正文、算法论文备份说明、写作过程稿和外部参考资料不放在本目录。

## 当前资料

- `API接口逐项说明_按docs顺序.md`：后端 API 中文辅助说明，对应 FastAPI `/docs`。
- `MySQL算法资源迁移说明.md`：Redis/SQL 双存储与 MySQL 迁移说明。
- `算法接入说明.md`：算法接入协议、审核口径和验收建议。

## 当前口径

- 项目主线是图像/视频增强复原任务的开放式算法性能测试平台。
- 项目不包含 LinUCB、多臂赌博机或 `POST /recommend/fast-select` 快速选型接口。
- 若接口或流程发生变化，应先以 `backend/app/main.py` 和前端页面实际行为为准，再同步更新本目录文档。
