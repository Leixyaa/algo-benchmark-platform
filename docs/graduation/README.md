## 口径提醒

- 当前项目主线是图像/视频增强复原任务的开放式算法性能测试平台。
- 当前项目不包含 LinUCB、多臂赌博机或 `POST /recommend/fast-select` 快速选型接口。
- 答辩材料应围绕统一评测流程、开放接入、异步执行、指标计算、对比导出、社区共享与后台审核展开。
