# 计算环境条件一验收清单

## 目标

证明系统至少具备两种计算环境：

1. Web 端
2. 桌面端

## Web 端验收点

- 能通过浏览器访问系统主页
- 能完成登录、数据集管理、算法管理、运行创建、结果对比、导出
- 前端入口位于 `web/`
- 浏览器访问地址为 `http://localhost:5173/`

## 桌面端验收点

- 能通过 `scripts/manual_up_desktop.ps1` 或 `scripts/manual_up_desktop.cmd` 启动
- 桌面端使用本地构建前端 `web/dist-desktop/`
- 桌面端默认使用本地 SQLite 文件 `backend/data/desktop/abp-desktop.db`
- 桌面端能启动本地后端 API 和 Celery Worker
- 桌面端能以窗口化形式访问核心模块
- 桌面端具备独立文档与交付包 `release/desktop-runtime/`

## 建议答辩演示顺序

1. 演示浏览器访问 Web 端主页
2. 演示 Web 端创建一次运行任务
3. 关闭浏览器页面，运行 `manual_up_desktop.cmd`
4. 演示桌面窗口打开系统首页
5. 展示桌面端仍可进入数据集、算法、运行、对比等核心模块
6. 展示桌面端默认 SQLite 数据文件位置
7. 展示 `release/desktop-runtime/` 目录作为桌面端交付物
